# AI-Powered FAQ Chatbot Portal

An intelligent full-stack FAQ chatbot web application that answers user queries using Google's Gemini AI API. The application provides real-time responses, stores chat history in MongoDB, and offers a responsive user-friendly interface built with React.

## Features

* AI-powered question answering using Gemini API
* Interactive chat interface
* Real-time query processing
* Chat history storage in MongoDB
* RESTful backend API with Express.js
* Responsive and modern UI
* Error handling and loading indicators
* Frontend and backend deployment support

## Tech Stack

### Frontend

* React.js
* Axios
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

### AI Integration

* Google Gemini API

## Project Structure

* Frontend: React-based chatbot interface
* Backend: Express API for handling chat requests
* Database: MongoDB for storing conversations
* AI Service: Gemini API for generating responses

## Installation

1. Clone the repository
2. Install dependencies for frontend and backend
3. Configure environment variables
4. Start MongoDB Atlas connection
5. Run backend server
6. Run frontend application

## Environment Variables

Create a `.env` file in the backend folder:

```env
GEMINI_API_KEY=your_gemini_api_key
MONGODB_URI=your_mongodb_connection_string
PORT=4000
```
